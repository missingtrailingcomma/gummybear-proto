# gummybear-proto

Protobuf for Gummybear entities and client and server communications.

## Usage

### Go

```sh
go get github.com/missingtrailingcomma/gummybear-proto@v0.0.2
```

### TypeScript

```sh
pnpm add github.com/missingtrailingcomma/gummybear-proto#v0.0.2
```

## Dev

```sh
make generate
```
